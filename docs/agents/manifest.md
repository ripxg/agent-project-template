## How to Register

Add a row when you start working:

```markdown
| claude-20250114-1430 | claude | active | 1-1 | 2025-01-14 14:30 |
```

**Agent ID format**: `[type]-[YYYYMMDD]-[HHMM]`

**Status values**:
- `active` - Currently working
- `idle` - Waiting for work/input
- `offline` - Session ended
